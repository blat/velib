Velib' for mobile
=====

Demo
-----------------

* [velib.blizzart.net](http://velib.blizzart.net/)


Dependances
-----------------

* sinatra
* xmlsimple
* open-uri


Setup
-----------------

1. Install ruby and all dependancies (see above)

2. Launch application:

        ruby velib.rb

3. Go to [http://localhost:4567](http://localhost:4567)
